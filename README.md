# Southpaw Financial Services

A premium React frontend for a modern management consulting & accounting firm.
Built with **Vite + React**, **React Router**, and **Framer Motion**.

## Pages
- **Home** — hero, stats, practice pillars, why-us, process, testimonial
- **About** — firm story, values, timeline, leadership
- **Our Solutions** — six service practices, analytics platform, sector expertise
- **Contact Us** — interactive enquiry form + global offices

## Design
- Deep-navy + warm-gold palette, ivory type
- `Fraunces` display serif + `Manrope` body (Google Fonts)
- Glassmorphism cards, ambient glows, film grain, scroll-reveal animations
- Fully responsive down to mobile

## Getting started

```bash
npm install
npm run dev      # start dev server at http://localhost:5173
npm run build    # production build
npm run preview  # preview production build
```

> The contact form is front-end only (simulated submission). Wire it to your
> backend/email service in `src/pages/Contact.jsx`.
